# Mi Galería Web - Desafío Front End (Nivel Experto 🚀)

Proyecto de galería web responsiva con diseño oscuro que consume datos en vivo directamente desde internet. Desarrollado para cumplir con la pauta de evaluación de la Unidad 1.

## 🔗 Enlaces de Entrega

- **Sitio en vivo (GitHub Pages):** [https://antoniofontanilla.github.io/mi-galeria-web/](https://antoniofontanilla.github.io/mi-galeria-web/)
- **Repositorio de Código:** [https://github.com/antoniofontanilla/mi-galeria-web](https://github.com/antoniofontanilla/mi-galeria-web)

## 🌐 API Utilizada

Se consume la API pública oficial **JSONPlaceholder** (Endpoint: `/users`), obteniendo información real de usuarios en formato JSON de forma asíncrona.

## 🏆 Características de Nivel Experto Implementadas

- **Buscador Funcional en Vivo:** Filtra dinámicamente las tarjetas en pantalla por nombre y ciudad a medida que el usuario escribe, manipulando el DOM en tiempo real.
- **Diseño Responsivo Avanzado:** Maquetación con CSS Grid que se adapta automáticamente a cualquier resolución (móvil y PC).
